# Lab-MovieTicket
Simple movie ticket code that lets you choose a valid seat, the price of the seat based on choice, and if the movie is evening or daytime
## How to run
1. Open IDE
2. Run the Program
3. Answer all questions asked
## Example Output
John Wick | Seat: 102 | Rating: R
Ticket price: $4
Evening Show
